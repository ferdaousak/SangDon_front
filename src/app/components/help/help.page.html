Help

Comment demander de sang Il suffit de se déplacer vers la page demander du sang et saisir le type du sang puis cliquer sur demander. Comment effectuer un don de sang La page d’acceuil contient une liste des demandes auxquelles vous pouvez appliquer. Comment consulter les centres de ma ville La page centre contient la liste des centres ,vous pouvez cliquer sur l’un des centres pour voir les détails (localisation, adress, numéro de téléphone ...) Comment prendre un rendez-vous La page rendez-vous permet de prendre un rendez-vous dans un centre.
